# 鸿蒙图书管理系统 java191

## 项目概述

鸿蒙图书管理系统是一款基于HarmonyOS的应用，旨在实现图书信息的管理，支持图书的增加、删除、修改和查询功能，同时具备用户登录和注册机制。

## 技术栈

- **前端**：使用ArkTs语言开发
- **后端**（鸿蒙+后端版本）：SpringBoot框架，MySQL数据库

## 功能模块

- 用户模块：登录、注册
- 图书管理：增删改查图书信息

## 系统角色

- 用户：可登录、注册、管理图书
- 管理员：具备所有用户权限，并能够管理用户信息（如需）

## 运行环境

- **纯鸿蒙版**：兼容HarmonyOS API 9
- **鸿蒙+后端版**：兼容HarmonyOS API 12，后端需部署在支持SpringBoot和MySQL的环境中

## 部署步骤

由于部署步骤涉及具体技术操作，以下仅概述：

1. 准备HarmonyOS开发环境
2. 克隆或下载项目代码
3. 配置后端环境（针对鸿蒙+后端版本）
   - 安装MySQL数据库
   - 部署SpringBoot后端服务
4. 编译和运行项目
